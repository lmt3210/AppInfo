Usage


Open AppInfo and wait for the list to be populated. Installed applications are initially displayed in alphabetical order by name. Clicking on any column header will allow sorting by the contents of the selected column.

Minimum OS:
15.xSequoia
14.xSonoma
13.xVentura
12.xMonterey
11.xBig Sur
10.15.xCatalina
10.14.xMojave
10.13.xHigh Sierra
N/AThe app does not report a minimum version


Possible architectures:
A64ARM 64-bit (Apple M1, M2, etc.)
I64Intel 64-bit
I32Intel 32-bit
P64PowerPC 64-bit
P32PowerPC 32-bit


The Info column displays the application's bundle information string if it is available, or the application's human-readable copyright string, if available. If neither is available (which is often the case with some applications), N/A is displayed.
